🚀 Introdução ao Back-end com Node.js
Este repositório contém meu primeiro servidor criado durante meus estudos de Back-end. O objetivo foi entender a base da arquitetura cliente-servidor e como as requisições web funcionam na prática.

🛠️ Tecnologias Utilizadas
Node.js: Ambiente de execução Javascript no servidor.

Express: Framework minimalista para gerenciar rotas e requisições.

NPM: Gerenciador de pacotes para instalação de dependências.

📖 O que foi praticado:
Configuração do ambiente de desenvolvimento.

Conceitos de arquitetura Cliente-Servidor.

Entendimento do Protocolo HTTP e seus verbos.

Criação de endpoints (rotas).

Estruturação básica de um projeto back-end.

🏁 Como rodar o projeto:
Clone o repositório.

No terminal, execute npm install para instalar as dependências.

Inicie o servidor com node server.js.

Acesse http://localhost:3000 no seu navegador.
